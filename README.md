# PatternFly Svelte

A Svelte component library implementing PatternFly design system components.

## Installation

```bash
npm install @patternfly/svelte @patternfly/patternfly
```

## Usage

```svelte
<script>
  import { Button } from '@patternfly/svelte';
  import '@patternfly/svelte/styles.css';
</script>

<Button variant="primary">Click me</Button>
```

## Development

### Setup

```bash
npm install
```

### For development run storybook

```bash
npm run storybook
```

### Build

```bash
npm run build
```

### Testing

```bash
# Run tests
npm test

# Run tests with UI
npm run test:ui

# Run tests with coverage
npm run test:coverage
```

### Storybook

```bash
# Start Storybook
npm run storybook

# Build Storybook
npm run build-storybook
```
